## 简介

这是一份 PyQt5 的教程。教程的目的是让你开始学会使用 PyQt5 工具包。

这个教程的内容在 Mac 上创建并测试。

## 安装

### 安装 qt

```bash
brew install qt
```

### pip install packages

```bash
pip install -r requirements.txt
```
